"My Gaming Collection" is an app for retro gamers who collects games, consoles and accessories. 
It's aim is to organize your collection, with pictures (these ones can be placed by the adming of the app... me) as they are located in a repo. 
Feel free to use your own implementation, or configuring it with your own google firebase account. 
This app makes use of Google Firebase's realtime database api, authentication api and storage api. 
Also makes use of of Awaitility for tests, JUnit4 and some others (check the gradle files for more info. 
It runs on Java 11 and target API is 34. Also uses new Jetpack Compose framework. 
Unit tests are done. Most of UI Tests are also done (this can be better by testing each screen separately) and Instrumentalization tests are not done yet.
